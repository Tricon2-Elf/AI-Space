mm-suite Evaluation Pack

mm-suite Evaluation Pack contains GMSV, SWP, and client for Windows.
You can try mm-suite with this package and backend, MSGSV which starts in www.ce-lab.net.

Compile the source (GMSV)

cd mmsuite_eval_xxxx_xx_xx/gmsv
make

Setting of GMSV

Please change the IP_ADDRESS in gmsv/data/fieldtab.txt to the address where you intend to start SWP.
For example, 192.168.1.203, you should change it as follows. 

8001 192.168.1.203 9001 land_0 dungeon_0
8003 192.168.1.203 9003 island_0

How to start GMSV

cd mmsuite_eval_xxxx_xx_xx/gmsv
make start_eval

The command above makes start two GMSV.
Please check like this.

porsche:/opt/chiaki/src % ps auxw|grep gmsv
96:chiaki 32534 5.6 3.2 69640 68060 pts/7 SN 19:11 0:00 ./gmsv gmsv_template.conf 8001.conf
98:chiaki 32537 5.3 3.2 70316 68068 pts/7 SN 19:11 0:00 ./gmsv gmsv_template.conf 8003.conf

How to start SWP

cd mmsuite_eval_xxxx_xx_xx/swp
make start

Two SWP start.

porsche:/opt/chiaki/src % ps auxw|grep swp
92:chiaki 32425 0.0 0.8 19696 16940 pts/5 SN 17:43 0:01 ../vce/swp 9001.conf swtable.9001.conf
94:chiaki 32428 0.0 0.8 18696 16936 pts/5 SN 17:43 0:00 ../vce/swp 9003.conf swtable.9003.conf

How to connect from client to GMSV

Please refer to wincli-en.html.
[ATTENTION!!] When you try the command, "msgconnect", please set [HOST] www.ce-lab.net.
