mm-suite is a development toolkit of Massive Multiplay Online Games. This document is describing how to develop/maintain mm-suite projects.

philosophy of mm-suite

Development of online games is now considered as a very difficult, expensive project. CommunityEngine and mm-suite has a final goal to reduce difficulty of online game development in straightforward ways:

  • create special libraries
  • create special tools for maintainance
  • set up a suitable development organization
  • instruct engineers

We believe that online game development can be more affordable by achieving these goals one by one.
