スタート位置定義ファイルの書式(startpos.c>

GMSV は、ユーザがキャラクターを作成したときに、 世界の中のどの位置から開始させるかを、「スタート位置定義ファイル」 の内容に基いて決める。 デフォルト状態では、 data ディレクトリの startpos.txt ファイルが スタート位置定義ファイルとなっている。 このファイルも1行に1設定となっている。 

cand=FIELDNAME,CATEGORY,X1,Y1,X2,Y2,Z
  • FIELDNAME
    フィールド名。 フィールド情報ファイルの name ディレクティブで 指定した、フィールド名。このフィールド内の矩形。
  • CATEGORY
    カテゴリ番号。 複数の矩形をグループ化して、 そのグループの中のどこかから出現ということを可能にするために、 カテゴリIDを指定できる。 カテゴリ ID が同じ設定はグループと見なされる。 カテゴリ ID は 1以上にすること。 0だとグループ化されない。
  • X1,Y1,X2,Y2
    この矩形の中のどこかが選択される。
