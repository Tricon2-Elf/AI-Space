bf
SummaryEncrypt many files by password.
Compatibilitylinux
Usagebf [-d] [-p password] infile infile ...
DescriptionThis tool encrypts an input file by using a password. The Blowfish algorithm is used for encryption. It is a block encryption in 8-bytes unit.
-d option decodes an encrypted file.
If-p option is not specified, it prompts you to input a password (getpasswd).
The original file is not deleted when its encryption is finished. The encrypted file will be saved with ".bf" added at the end of original file name. To decrypt an encrypted file, the same password used for encryption is required.
Example
bf somefile.txt somefile2.txt
   (enter a password at a terminal and do encryption)
bf -d somefile.txt.bf somefile2.txt.bf
   (enter a password at a terminal and do decryption)
Bugs/TODOTODO Since it enciphers after extracting the contents of a file into a memory, the file wouldn't be encrypted if the file is too big for total amount of available memory.


hextext
SummaryConvert binary files into VCE-hex-text format.
Compatibilitylinux
Usagehextext [-d] [-v] infile infile infile ...
DescriptionThis was originally created to test the vce_make_hex_text_from_buffer function.
-d If this option is added decoding is done, and if it is omitted encoding is carried out. You can convert as many files as you want to at one time. When conversion is successful, a new file with ".hextext" appended after the input file name is created as in "INFILE.hextext". The source file is not deleted. When a hextext format file with a name of "hoge.text" is decoded successfully, the file named "hoge" will be overwritten, so caution is necessary. In addition, when the -v option is specified, the version information is displayed and a normal exit occurs.
Example
hextext a.txt b.txt
Bugs/TODOhextext loads all files into RAM in the first step of encoding, so you cannot encode/decode large files that is too large to be loaded on RAM. The maximum size of input files is about 10 or 100MBytes. ( It's dependent on the amount of RAM put in the machine.)
