libvce.a
SummaryVCE for Linux static-link library
Compatibilitylinux,darwin
Descriptionvce.h includes a prototype declaration and an extern variable declaration for all functions of VCEgg API.
Notes for compiling under a C++ environment A compiler must be specified to define _the cplusplus macro. VCEgg's code is essentially written with UNIX's C so it has the "linkage" of C. Microsoft VisualStudio, for instance, has "C++ linkage" as default, so a prototype declaration by extern "C" is required to use functions with a C linkage.

vcewin32.lib
SummaryVCE for Windows 98/NT/2000 static-link library file
Compatibilitywin32
Descriptionvcewin32.lib is the library file for Win32 environment. VisualStudio 6.0 is used for compilation. Basically it includes most of the VCE's functions.
The file name of vcewin32.lib in the VCE package is 
vcewin32_1_0_0.lib
vcewin32_1_0_0mt.lib
vcewin32_1_0_0mfc.lib
It is recommended not to change this file name as much as possible to avoid confusion on the versions. If you do not wish to change the file name for a development project, simply create a Window shortcut. The file including "mt" in the file name is the library to link with a multi-thread version of the C library.
