admcli: words

    admsv
    Special server program for monitoring server process and server machines. Only one admsv is running in one domain. admcli connects to this process. admsv will connect to rsv, swp, and every server programs which you want to monitor.
    rsv
    Runs on every target MACHINE. Admsv will connect to every rsv and sends command, and obtains host information from rsv.
    process
    process will be identified by IP address and monitoring port number. Note that service port number is not used for identification. In this document, we will often use "process" as "process that is monitored by admsv".
    host
    host will be identified by IP address. In this document, we will often use "host" as "host that is monitored by admsv".
    VCE monitor API
    A set of API and protocol implemented in vce. This simple protocol is used to monitor remote server activity.
    Monitor command
    commands defined in VCE monitor API.
