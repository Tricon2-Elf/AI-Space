admcli installation

  1. Expand admcli.zip in some directory
  2. Execute setup.bat in expanded directory

    If you cannot execute it, please try DockStudioXP out, it can be obtained from http://www.innovasys.net.

  3. After all, you can start admcli program. You will execute "admcli.exe".
