Log search window

You can search through log file grep-like. This search can only support log file formatted as defined by Admcli Standard Log Format.

Description of Log search window

Search user NAME
Find NAME of user. Requires full match. (for example "rin" doesn't match "ringo" )
Search user ID
Find ID of user. Requires full match
Search any string
Find any string in log file. This looks at "free text" in standard log format .
Target process(required)
Determine target process.
Type(required)
Select type from dropdown list in combo-box.
Time
You can specify From and To. By omitting From, it starts from the beginning. By omitting To, it ends at the last line.
Transfer log using ZIP compression
Compress log file by ZIP and save it in local file.
Realtime search
This behaves like "tail -f". When new log is comming in, it will be searched and shown in this window.
Start Search
By clicking this button, it starts searching and result will be shown in this window.
Merge
Sort multiple search result by time, and show it in new window.
