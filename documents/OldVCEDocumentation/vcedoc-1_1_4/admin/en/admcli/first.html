What to do first

You have to perform some configuration on admcli before you use it first.


  • configure user ID
  • configure network information

configure user ID

(1) Select [Tool]-[Configuration] menu, and you will see a dialog. Then select "Network" in that dialog tree view.
(2) Input user ID and password of yours. You may have to ask admsv maintainer about your information.

configure network

(1) Select [Tool]-[Configuration] Menu, and you will see a dialog. Then select "Network" in that dialog tree view.
(2) Input admsv server IP address(or DNS name), and port number. You may have to ask admsv maintainer about it.
After finishing these configuration, you will press "OK" button and close config dialog. After all you can connect to admsv by selecting [File] - [Login] from menu.
