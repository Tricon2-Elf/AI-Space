Configuration

You can preform precise confiuration against admcli by selecting [Tool]-[Configuration] menu dialog.

General configuration

Line number of log window buffer
Configure number of text lines of system log and result log. It can lead high CPU usage by setting too much line. Default value is 500.

Network configuration

user ID
set user ID to log in to admsv .
user password
ser user password to log in to admsv.
user nickname
set your nickname when chatting through admsv.
auto login
admcli automatically log in to admsv if you set this option "ON".
auto reconnect
admcli automatically reconnects to admsv if the connection is shut-down by some reason during monitoring, by setting this option "ON".
admsv server address
set server address of target admsv. admcli tries to connect to this address.
admsv port number
set port number of admsv
interval of network polling
set interval time to poll the server. The shorter, the quicker response.

Log

Directory name of system log
set pathname of system log directory.
Prefix of system log file
admcli uses this prefix when saving and rotating system log in the directory.
System log file will be named like: "DIRNAME\PREFIX_YYYYMMDD.txt"
Directory name of result log
set pathname of result log directory.
Prefix of result log files
admcli uses this prefix when saving result log in the directory. It uses same rule as "system log file".
Direcotry name of alert log
set pathname of alert log directory.
Prefix of alert log files
admcli uses this preifx when saving alert log in the direcotory. same rule as "system log".
Directory name of graph-log
set pathname of graph log directory.
Prefix of graph log files
admcli uses this prefix when saving graph log. Graph log file will be named as "DIRNAME\PREFIX_identifier_type_id_YYYYMMDD.txt" .
For example, log of HOST-ping line will be saved in file: "graphlog/graphlog_host_ping_1_20030811.txt"

Graph

Number of entries of saved information of graph
Width of each graph. Bigger number, longer graph.
Min and Max and auto value of each graphd
Minimum value is the bottom line of each graph. If the data is less than this value, it will never plotted in graph. Maximum value is the upper-top line of each graph.

Font

You can choose any font in these admcli windows :


  • HOST window
  • Process window
  • Log window
  • Result window
  • Search window
  • Chat list window
  • Chat window

    Alert

    By setting alert, admcli shows "!" mark in process window and host window's "alert" column, and the line gets yellow. You can make admcli to reproduce alert sounds or execute external application program.
