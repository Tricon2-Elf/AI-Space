VCE admin tools 内で使われる単語について

VCE admin tools 内で使われる単語のなかで、特殊な意味を持つものについて説明します。

プロセス

通常のプロセスを指すこともありますが、 主にVCE monitor APIによって管理できるプロセスを指します。
プロセスは、IPアドレスと監視用のポート番号で一意に区別されます。
サービス用のポート番号ではないことに注意して下さい。

ホスト

通常のホストを指すこともありますが、 主にrsvによって管理されているホストを指します。
ホストは、IPアドレスで一意に区別されます。

モニターコマンド

VCE monitor APIを通して実行可能なコマンドです。詳しくは、 monitor commandを参照してください。

UNIXコマンド

通常のUNIXコマンドラインから実行可能なコマンドです。 rsvからは、UNIXコマンドが実行可能です。
