用語集

    admsv
    サーバー管理の為のサーバー。admcli クライアントがネットワーク接続をする。 rsv や swp、モニタリングしたいサーバーなどと接続し、情報取得、操作などクライアントからの要求をrsvに送る事などをする。
    rsv
    各サーバーにおいて起動され、admsvから要求されたサーバーの情報の伝達、コマンドの実行などを行うサーバー。
    プロセス
    通常のプロセスを指すこともありますが、 主にVCE monitor APIによって管理できるプロセスを指します。 プロセスは、IPアドレスと>監視用のポート番号で一意に区別されます。 サービス用のポート番号ではないことに注意して下さい。
    ホスト
    通常のホストを指すこともありますが、 主にrsvによって管理されているホストを指します。 ホストは、IPアドレスで一意に区別されます。
    VCE monitor API
    vceに組み込まれた、VCE monitor API。主に vce アプリケーションの情報の伝達、操作をする為に使用される。VCE monitor API に反応するようにアプリケーションをプログラミングする事によって、admin tool などで VCE アプリケーションの情報の閲覧、操作などを行う事が出来る。
    モニターコマンド
    VCE monitor APIを通して実行可能なコマンドです。
