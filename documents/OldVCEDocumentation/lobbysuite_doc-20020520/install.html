lobby-sutieプログラム群のコンパイル

開発マシンへのインストール

開発用マシンは，最低1台必要です。lobby-suiteはlinuxでの動作を確認しています。 gcc, glibc, kernel はできるだけ最新のものが良いでしょう。

必要なもの

  • Ruby インタプリタ <URL:http://www.ruby-lang.org/> からダウンロード出来ます。
  • VCE (出来るだけ最新のもの)

サーバープログラムのコンパイルに必要なもの

  • libmysqlclint.so (MySQLをコンパイル，インストールする)

クライアントプログラムのコンパイルに必要なもの

SDLライブラリー
linux/win32 で動作させる場合のみ必要です。 <URL:http://www.devolution.com/~slouken/SDL/> からダウンロード出来ます。
PlayStation2開発キット
PlayStation2で動作させる場合のみ必要です。

コンパイルするには

lobby-suiteサーバー群をコンパイルするには，lobbysuiteのトップディレクトリで，

% make

とするだけである。これにより，lobbysv/authsv/dbsv/locksv/msgsv/tester/cli 全てのプログラムがコンパイルされることになります。

クライアントプログラムのコンパイル

lobbysuite/cli/ ディレクトリで，

% make

コマンドを実行すれば，PlayStation2用elfバイナリが作成されます。 linux用クライアントを作成するには，

% make linux

としてください。
